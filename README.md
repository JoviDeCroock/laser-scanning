# laser-scanning

EDS Systems marketing site built with Astro and React.

## Development

- `yarn dev` starts local development server
- `yarn build` creates a production build in `dist`
- `yarn preview` previews the production build locally

## Environment variables

- `PRISMIC_REPOSITORY`
- `PRISMIC_ACCESS_TOKEN`
- `SENDGRID_API_KEY`
- `EMAIL_DESTINATION`
- `EMAIL_SENDER`
